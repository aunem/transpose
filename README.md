# Transpose

Transpose is a cloud native composable proxy with a focus on Kubernetes written in Go.

* NOTE: Transpose is pre alpha and under heavy development

## Tenants

* Composable
* Observable
* Simple
* Lightwieght
* Developer friendly
* Cloud native
